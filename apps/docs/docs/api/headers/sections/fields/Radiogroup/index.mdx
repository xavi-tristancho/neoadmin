The Radiogroup is a type of `field`. Is the representation of a group of radio buttons in which only one option can be selected at a time.
When defining a `field` of type `radiogroup` extra properties need to be defined. Such as `options`.
You can control its representation with some extra properties either:

### Example

```js
import { CustomFieldsContainer } from "components";

const header = {
  type: "CRUD",
  options: {...},
  sections: [
    {
      fields: [
        // highlight-start
        {
          property: "state",
          type: "radiogroup",
          options: [
            { label: "Published", value: "published" },
            { label: "Unpublished", value: "unpublished" },
            { label: "In Review", value: "in_review" },
          ]
        },
      // highlight-end
      ],
    }
  ],
};

export default headers;
```
