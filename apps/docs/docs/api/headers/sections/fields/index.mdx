`fields` is an array of objects. Each object is meant to be a `field` that you specify what is needed to show in the Table and/or the Form with its intrinsic properties. This property is located in the root of every Section and has different variants.

### Example

```js
import { CustomFieldsContainer } from "components";

const header = {
  type: "CRUD",
  options: {...},
  sections: [
    {
      // highlight-start
      fields: [
        {
          label: "Image",
          property: "image",
          type: "image",
        },
        {
          label: "Title",
          property: "title",
          type: "text",
        },
        {
          label: "Date",
          property: "date",
          type: "date",
        },
        {
          label: "Content",
          property: "content",
          type: "html",
        },
        {
          label: "Category",
          property: "category",
          type: "multiselect",
          relation: {
            isMulti: false,
            options: categories,
            format: (category: Category) => category.name,
          },
          tableOptions: {
            format: (category: Category) => category.name,
          },
        },
        {
          property: "visible",
          tableOptions: {
            show: false,
          },
          type: "checkbox",
        },
      ],
      // highlight-end
      component: () => {
        return <div>This is a section custom component</div>;
      },
    }
  ],
};

export default headers;
```

## Table Options

The `tableOptions` object allows you to specify and customize how each field will be shown at the table.

```js
const headers = {
  type: "CRUD",
  options: {...},
  sections: [
    {
      fields: [
        {
          label: "Title",
          property: "title",
          type: "text",
          // highlight-start
          tableOptions: {
            show: true,
            format: ({ state }) => <div>{state.title}</div>,
          }
          // highlight-end
        },
        {
          label: "Author",
          property: "author",
          type: "text",
          // highlight-start
          tableOptions: {
            show: (state) => !state.title && state.title === "",
          }
          // highlight-end
        },
      ],
    }
  ],
};

export default headers;
```

## Upsert Options

The `upsertOptions` object allows you to specify and customize how each field will be shown and also alter/handle its value in the form.

```js
const headers = {
  type: "CRUD",
  options: {...},
  sections: [
    {
      fields: [
        {
          label: "Title",
          property: "title",
          type: "text",
          // highlight-start
          upsertOptions: {
            show: true,
            component: ({ state }) => <div>{state.title}</div>,
          }
          // highlight-end
        },
        {
          label: "Author",
          property: "author",
          type: "text",
          // highlight-start
          upsertOptions: {
            show: (state) => !state.title && state.title === "",
          }
          // highlight-end
        },
      ],
    }
  ],
};

export default headers;
```
